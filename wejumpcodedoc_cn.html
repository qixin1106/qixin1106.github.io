如何添加自定义训练

添加自定义训练(操作流程)

  1. 首先您需要一份正确的训练配置文件(JSON)
  2. 将配置文件复制到手机剪切板中
  3. 重新打开WeJump
  4. 如果前面步骤正确,您将会看到WeJump识别出了新的训练动作的页面
  5. 点击保存
  6. 到训练/训练列表/自定义列表中查看

通俗的理解为淘宝口令形式,希望有助于您理解.

代码说明

示例

  • 使用 JSON 的自定义训练配置文件
  • 你需要确保配置文件的格式正确.
{
  "descs" : "训练项目说明",
  "level" : 0,
  "name" : "训练项目名字",
  "items" : [
    {
      "bpm" : 0,
      "name" : "准备",
      "type" : 2,
      "second" : 10
    },
    {
      "bpm" : 160,
      "name" : "30秒单摇",
      "type" : 0,
      "second" : 30
    },
    {
      "bpm" : 0,
      "name" : "休息",
      "type" : 1,
      "second" : 30
    },
    {
      "bpm" : 200,
      "name" : "30秒双摇",
      "type" : 0,
      "second" : 30
    }
  ]
}

参数说明

  • name : String

训练的名称，限制为 30 个字符，如果超过将被截断。

￼

  • descs : String

训练项目说明，限140个字符，超过140个字符将被截断。

我们建议您在描述中提及有价值的信息，例如 bpm 或训练强度。

￼

  • level : Int

训练级别是根据您的喜好定义的。 当然，您可以将非常简单的训练定义为“困难”。 但是我们希望您认真定义级别。 当你把训练代码分享给别人的时候，别人可以根据级别理解这个训练。

我们有5个训练级别，如果您设置的级别不在此范围内，我们将默认为"简单"。

0 is "简单"
1 is "中等"
2 is "困难"
3 is "英雄"
4 is "传说"

￼

  • items : Array

这是您的训练计划的动作列表,其中每个动作由以下数据构成

{
      "bpm" : 0,
      "name" : "Ready",
      "type" : 2,
      "second" : 10
}
  • name : String

这是您的训练动作的名称，限制为 30 个字符，如果超过它将被截断。

￼

  • second : Int

每个动作的持续时间（以秒为单位），最小值为 10秒，最大值为 1800秒（30 分钟），如果不在此范围内我们将设置为最接近的有效值。

￼

  • bpm : Int

bpm 值是每分钟的跳跃次数。 例如，如果 bpm 等于 160，则意味着每分钟需要 160 次单摇。 当然，如果是双摇，您可以根据跳的次数来设置bpm，也可以根据绳子的旋转来设置bpm，例如1分钟100个双摇，您可以设置bpm为200，你可以在节拍器页面来体验bpm的实际速度。

最小值为 60，最大值为 500。如果您设置的值不在此范围内，则默认设置为最接近的有效值。

bpm在 type = 1 或者 type = 2 时,设置无效.默认写0即可,请参考示例.

  • type : Int

我们有以下3个类型

0 is 动作 (我们一般使用"0",来做为跳绳训练的动作)
1 is 休息
2 is 准备

"休息"和"准备"的区别，我们希望你在开始训练的时候有一个准备时间，比如点击开始训练，你需要放下手机，然后拿起跳绳，我们推荐你的动作清单 ，第一个是"准备"。

不同的类型将用不同的颜色标记。

"休息"和"准备"类型时,节拍器提示无效

￼
